# Папка для документов ГОСТ

## Как добавить документы ГОСТ

### Вариант 1: Скачать с официальных источников

1. **Росстандарт (official)** - https://rskg.ru/
2. **Техэксперт** - https://techexpert.pro/
3. **КонсультантПлюс** - https://www.consultant.ru/
4. **Гарант** - https://www.garant.ru/

### Вариант 2: Использовать онлайн-ресурсы (быстро)

Перейдите на сайт и скачайте PDF:
- https://docs.cntd.ru/ - введите номер ГОСТа в поиск
- https://allgosts.ru/ - бесплатный архив ГОСТов

### Формат именования файлов

Файлы должны именоваться по следующему формату:
```
gost-{номер}-{год}.pdf
```

Примеры:
- `gost-1050-2013.pdf`
- `gost-8239-89.pdf`
- `gost-14080-78.pdf`

⚠️ **Важно:** Используйте строчные буквы и дефисы вместо точек!

## Список необходимых ГОСТов для системы

См. файл `gost_links.txt` в корне проекта для скачивания основных ГОСТов.

## Интеграция локальных файлов

Система автоматически использует локальные файлы из этой папки.
При нажатии на ГОСТ в карточке материала или в разделе "Документы" 
откроется локальный PDF-файл, если он существует.

Если файл не найден, браузер покажет ошибку 404 - в этом случае 
нужно скачать соответствующий ГОСТ и сохранить в эту папку.
