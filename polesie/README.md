# Система управления производством ОАО "Полесьеэлектромаш"

## 📋 Описание проекта

Профессиональная веб-система для автоматизации управления производством электротехнической продукции. Предназначена для полного цикла управления предприятием: от приёма заказов до контроля качества и отгрузки готовой продукции.

### 🎯 Основные возможности

- **Управление заказами** - создание, редактирование, отслеживание статусов заказов
- **Производственное планирование** - формирование производственных заданий, контроль этапов
- **Контроль качества (ОТК)** - многоступенчатая система проверок качества
- **Складской учёт** - учёт материалов, сырья и готовой продукции
- **Управление контрагентами** - база заказчиков и поставщиков
- **Отчётность и аналитика** - различные отчёты по производству и продажам
- **Разграничение прав доступа** - 7 ролей пользователей с разными правами

### 👥 Роли пользователей

1. **Администратор** - полный доступ ко всем функциям
2. **Менеджер по продажам** - управление заказами и клиентами
3. **Технолог** - управление производственными процессами
4. **Инспектор ОТК** - контроль качества продукции
5. **Кладовщик** - управление складом
6. **Производственный рабочий** - выполнение заданий
7. **Руководитель** - просмотр отчётов и аналитики

## 🛠️ Технологии

- **Backend**: PHP 7.4+
- **Database**: MySQL 5.7+ / MariaDB
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Server**: XAMPP / Apache
- **Design**: Современный адаптивный интерфейс

## 🚀 Установка и запуск

### Шаг 1: Установка XAMPP

1. Скачайте XAMPP с официального сайта: https://www.apachefriends.org/
2. Установите XAMPP в директорию по умолчанию (C:\xampp или /opt/lampp)
3. Запустите Apache и MySQL через панель управления XAMPP

### Шаг 2: Развёртывание проекта

1. Скопируйте папку `polesie` в директорию `htdocs`:
   - Windows: `C:\xampp\htdocs\polesie`
   - Linux: `/opt/lampp/htdocs/polesie`

### Шаг 3: Создание базы данных

1. Откройте phpMyAdmin: http://localhost/phpmyadmin
2. Создайте новую базу данных или импортируйте готовую:
   - Нажмите "Импорт"
   - Выберите файл `polesie/sql/database.sql`
   - Нажмите "Выполнить"

Или выполните SQL через консоль:
```bash
mysql -u root -p < polesie/sql/database.sql
```

### Шаг 4: Настройка подключения

Откройте файл `config/config.php` и при необходимости измените параметры подключения:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'polesie_production');
define('DB_USER', 'root');
define('DB_PASS', '');  // По умолчанию пустой пароль
```

### Шаг 5: Первый вход

Откройте браузер и перейдите по адресу: **http://localhost/polesie**

## 📊 Функциональные модули

### 1. Управление заказами
- Создание новых заказов с автоматической генерацией номера
- Добавление позиций заказа с выбором из каталога продукции
- Расчёт стоимости с учётом скидок
- Отслеживание статусов заказа
- Печать документов по заказу

### 2. Производство
- Формирование производственных заданий на основе заказов
- Планирование этапов производства
- Назначение ответственных и бригад
- Контроль выполнения заданий
- Учёт технологических карт

### 3. Контроль качества
- Многоуровневая система проверок
- Фиксация дефектов и несоответствий
- Фотофиксация результатов проверок
- Отчёты по качеству продукции
- Управление доработками

### 4. Склад
- Учёт материалов и сырья
- Движение товаров (приход/расход/перемещение)
- Контроль остатков
- Инвентаризация
- Учёт готовой продукции

### 5. Отчётность
- Отчёты по производству
- Анализ продаж
- Статистика по качеству
- Движение денежных средств
- KPI сотрудников

## 🔒 Безопасность

- Хэширование паролей (bcrypt)

- HTTPS поддержка
