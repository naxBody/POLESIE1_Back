# Инструкция: Печать паспорта с предварительным редактированием

## Описание
При нажатии кнопки "🖨️ Печать" на странице паспорта изделия теперь открывается модальное окно, где можно отредактировать все данные перед печатью.

## Как использовать

### 1. Открытие предпросмотра
- Откройте страницу паспорта изделия (Production → Серийные номера → Просмотр паспорта)
- Нажмите кнопку **"🖨️ Печать"** в верхней панели

### 2. Редактирование данных
В открывшемся модальном окне доступны для редактирования:

#### 📅 Даты и гарантия
- **Дата изготовления** - дата производства изделия
- **Гарантийный срок (мес.)** - продолжительность гарантии в месяцах
- **Начало гарантии** - дата начала гарантийного периода
- **Окончание гарантии** - дата окончания гарантии

#### 🏢 Данные организации
- **Название организации** - полное наименование производителя
- **Адрес** - юридический адрес организации
- **Телефон** - контактный телефон
- **E-mail** - адрес электронной почты

#### 📦 Информация об изделии
- **Наименование изделия (кастомное)** - можно переопределить стандартное название
- **Описание изделия** - дополнительное описание для паспорта

### 3. Сохранение и печать
- После внесения изменений нажмите кнопку **"💾 Сохранить и печатать"**
- Данные будут сохранены в базу данных (таблица `passport_dynamic_data`)
- В новой вкладке автоматически откроется версия паспорта для печати с обновлёнными данными

## Хранение данных

Все отредактированные данные сохраняются в таблице БД `passport_dynamic_data`:
- Связь с серийным номером через `serial_number_id`
- При повторной печати загружаются последние сохранённые значения
- Если данные не заполнены, используются значения по умолчанию из карточки изделия

## Структура таблицы passport_dynamic_data

```sql
CREATE TABLE `passport_dynamic_data` (
  `id` INT AUTO_INCREMENT PRIMARY KEY,
  `serial_number_id` INT NOT NULL UNIQUE,
  `warranty_start` DATE,
  `warranty_end` DATE,
  `warranty_months` INT DEFAULT 12,
  `manufacture_date` DATE,
  `product_name_custom` VARCHAR(255),
  `product_description` TEXT,
  `company_name` VARCHAR(255),
  `company_address` VARCHAR(500),
  `company_phone` VARCHAR(50),
  `company_email` VARCHAR(100),
  `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  `updated_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

## Примечания

1. **Обратная совместимость**: Если для серийного номера нет записей в `passport_dynamic_data`, используются стандартные данные из карточки продукта
2. **Приоритет данных**: Кастомные данные → Стандартные данные → Значения по умолчанию
3. **История версий**: При каждом сохранении создаётся новая версия в таблице `product_passport_versions`
4. **Права доступа**: Редактирование доступно пользователям с правом `production.edit`
